# DD Planejamentos e Resultados — Landing Page

Landing page estática (HTML + CSS) para a DD Planejamentos e Resultados, com foco em pequenos negócios locais: implantação do zero (DD Start) e organização/crescimento em 90 dias (DD Essencial 90 e DD Escala 90 + 6).

## Visão geral
Este repositório contém um site simples e funcional, sem dependências externas, ideal para publicação no **GitHub Pages**.

## Estrutura do projeto
- `index.html` — página única do site (HTML + CSS embutido)
- (opcional) `assets/` — imagens, ícones, arquivos estáticos
